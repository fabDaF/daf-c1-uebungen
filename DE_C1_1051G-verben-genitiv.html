📘 Die Regel: Verben mit Genitiv-Ergänzung

  • Eine kleine, aber wichtige Gruppe von Verben regiert im Deutschen den Genitiv. Sie sind typisch für gehobene Schriftsprache.
  • sich bedienen + Gen. → etw. nutzen/einsetzen: sich modernster Methoden bedienen
  • gedenken + Gen. → feierlich an etw. erinnern: der Gefallenen gedenken
  • sich enthalten + Gen. → auf etw. verzichten (auch: Enthaltung bei Abstimmung): sich jedes Kommentars enthalten
  • sich entledigen + Gen. → etw. loswerden / eine Aufgabe erledigen: sich einer lästigen Pflicht entledigen
  • bedürfen + Gen. → benötigen / erfordern (formell): einer Genehmigung bedürfen
  • sich annehmen + Gen. → sich um etw./jmdn. kümmern: sich eines Falles annehmen
  • sich rühmen + Gen. → etw. als eigene Leistung darstellen: sich eines Erfolgs rühmen
  • ermangeln + Gen. → an etw. fehlen lassen: jeder Grundlage ermangeln
Merkhilfe: Im Alltag oft mit Akkusativ ersetzt (z.B. etwas brauchen statt einer Sache bedürfen). In formellen Texten ist der Genitiv jedoch stilistisch geboten.
